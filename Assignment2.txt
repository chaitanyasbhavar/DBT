Q.5 Why isn't it possible to see the first five rows of a table?
Ans: Row's inside a table are scattered all over the database server hard disk when we select from a table ,the order of rows in the output ,depends on the row address,
     it will always be in ascending order of row address.Hence it is not possible to see the first 'five' rows of a table or last 'Five' rows of a table.
